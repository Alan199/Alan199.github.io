《暗时间》笔记

Veröffentlicht am 2017-06-10
  • “暗时间”：走路，买菜，洗脸，洗手，坐公车，逛街，出游，吃饭，睡觉，所有这些时间都可以成为暗时间，充分利用这些时间进行思考，反刍和消化平时看和读的东西
  • 思考得越多的人，心理年龄越大
  • 能够迅速进入专注状态，以及能够长期保持专注状态是高效学习的两个最重要习惯
  • 抗干扰能力：经常利用碎片化时间阅读和思考，对迅速集中注意力非常有帮助
  • 善于做规划的人，将目标分割成一个个里程碑，再将里程碑分割成TODO列表。对目标几个重大步骤有清晰的界定，每个步骤的耗时作出靠谱的上界估计

  • 你所拥有的知识并不取决于你记多少，而在于它们能否在恰当的时间被回忆起来：

    知识自动蹦出来：

    • 养成习惯，经常主动回顾一段时间学到的东西
    • 创造回忆的机会：
      • 经常与别人讨论，或者讲给别人听
      • 经常整理笔记并养成记笔记的习惯
      • 书写：将一段时间学习的知识按照一个主题系统地“串”起来，平添无数提取线索
    • 设身处地地“虚拟经历”别人经历过的事
    • 联系/比较自身的经历

作者的学习习惯

  • 学习与思考
    • Google&Wiki:遇到问题做的第一件事，也是学习某个东西做功课时最先用到的东西
    • 看书挑剔，只看经典
    • 做读书笔记。一是将阅读的思考及闪念写下来，二是将书中的好例子摘抄下来。人与人的差距不在谈资，而是花在思考的时间和思考的深度上
    • 利用走路和吃饭的时间思考，还有睡觉前必然要弄一个问题放在脑子里，在思考中入睡
      • 思考还能避免焦虑
    • 重要的事优先，尽量避免琐事干扰
    • 重要的事营造比较大的时机块来完成
    • 多看心理和思维的书
    • 学习一项知识，问自己三个问题：1.它的本质是什么2.它的第一原则是什么3.它的知识结构是怎样
    • 改变拖延的习惯：承认它很难改、要在过程中常常观察自己的行为
  • 学习和思考的过程中常问自己的几个问题
    • 你的问题到底是什么？（提醒自己思考不要偏离问题）
    • 到现在为止我收获了什么？（提醒自己整理学习的东西，时不时去总结）
    • 设想自己正在将东西讲给别人听
    • 时常反省和注意自己的思维过程。遇到无法解决的问题，将原先的思维过程回顾一遍，看看到底哪个环节被阻塞了妨碍了理解。将思维过程写下来
    • 养成反驳自己的想法的习惯
    • 当理解了一个问题，问问自己真的理解了吗？对这个理解满意吗？发散思维
  • 重视知识的本质：抓住不变量，程序员行业大量的新技术，其实只是一层皮，背后支撑的东西是十年来不变的东西
  • 重视积累的强大力量
  • 时不时审视自己正在做的事情对现在和未来有什么价值，是不是自己真正希望做到的，学到的东西重要吗

送给自己的话

1.一生的知识积累，自学的起码占90%

2.遇到任何问题，找书，谷歌，wiki，百度

3.怪自己，不怪书，不怪老师，不怪环境

第一篇博客，献给我写博客的心酸历程

以下是大佬分享的写博客教程:

$ hexo new “文件名”

写markdown需要用到一些语法：

可以参考Cmd Markdown简明语法手册

插入图片时遇到了小麻烦。百度了一下，有两种方法可以插入本地图片：

第一种，上传本地的：

CodeFalling/hexo-asset-image的插件来加载本地图片：

1.首先确认_config.yml 中有 post asset folder :true。
Hexo 提供了一种更方便管理 Asset 的设定：post_asset_folder
当设置post_asset_folder为true参数后，在建立文件时，Hexo
会自动建立一个与文章同名的文件夹，可以把与该文章相关的所有资源都放到那个文件夹，如此一来，便可以更方便的使用资源。

2.在hexo的目录下执行npm install https: //github.com/CodeFalling/hexo-asset-image –save（需要等待一段时间）。

3.完成安装后用hexo新建文章的时候会发现_posts目录下面会多出一个和文章名字一样的文件夹。图片就可以放在文件夹下面。

第二种，远程地址：

申请一个七牛云账号，把图片存储到云，然后复制外链就可以得到地址

或者先上传到github仓库再去调用，使用远程地址

然后又遇到了图片太大不整洁的问题：

图片大小解决方法：

我使用七牛云存储图片，所以通过七牛云图片样式改变大小：
先建立图片样式，然后在外链中加样式接口
例如(http://ooupgqdn7.bkt.clouddn.com/QQ%E5%9B%BE%E7%89%8720170425215512.png-450300)
-450300是样式接口，手动加上去

最后再分享一款写markdown的软件:typora
感谢小旋旋的推荐，这款软件可以直接预览效果，且页面简洁，比用word直接敲好用了不知多少倍
