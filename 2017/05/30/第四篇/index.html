Q:hexo换了主题之后只有文字

背景：就是看中了一款主题，迫不及待去换。结果问题来了！
主题主人(不知道是不是)说要安装渲染器，嗯，网络原因安装不成功。通过淘宝镜像安装后，可以了。BUT，还是只有文字
果然还是要专一，手残专业户真是伤不起

原因：主题下载不成功

但是再次下载主题出现了这样一个错误：fatal: destination path ‘themes/maupassant’ already exists and is not an empty directory.

再换另一个主题就0K了。如果还想用一开始相中的，那再次换主题吧。
